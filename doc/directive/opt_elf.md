Asmc Macro Assembler Reference

### OPTION ELF

**OPTION ELF**:_Value_

Allows to fine-tune the values written to the ELF header if output format ELF.

    OPTION ELF:osabi

The only argument _osabi_ will be copied to the ELF header field EI_OSABI.
